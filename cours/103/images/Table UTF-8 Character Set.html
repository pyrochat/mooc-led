Table UTF-8 Unicode Character Set - for HTML UTF-8 enabled pages.

Working in a bi/multi-lingual environment can be a challenge when coding HTML pages with non-standard characters.
I try to find the simplest solution that works for me, and stick to it!  This is how I've solved this one.
Using the META tag <meta http-equiv=Content-Type content="text/html; charset=UTF-8"> I use the code table below for all non standard characters.
